# CostumeTrack

Inventory management and rental tracking for costume shops, theaters, and costume makers.

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components
- **Prisma** - Database ORM
- **PostgreSQL** - Database (via Neon)
- **NextAuth.js** - Authentication

## Features

- Multi-tenant organization support
- Costume inventory management with photos
- Rental tracking with check-out/check-in workflow
- Customer and production management
- Overdue rental alerts
- Search and filtering

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
